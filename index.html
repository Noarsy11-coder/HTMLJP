Preguntas de CSS (Q&A)

  1. Como ingeniero de software, ¿recomiendas usar !important? Solo como último recurso. Puede romper la cascada y volver el CSS difícil de mantener. Mejor arreglar especificidad, orden o estructura de clases.
  2. Si pones una imagen de fondo en HTML, ¿por qué debes escogerla con cuidado? Por rendimiento (peso), legibilidad del texto, tamaños/resolución, y derechos de uso. Ideal: comprimida, con buen contraste y en formato moderno (WebP).
  3. ¿Cómo eliges unidades entre %, px y pt? % para layouts fluidos, px para control fino (con cuidado en responsive) y pt casi nunca en web (más para impresión). En web también conviene usar rem para tipografías.
  4. ¿Por qué CSS minificado mejora rendimiento? Reduce tamaño del archivo → baja tiempo de descarga y parsing → carga más rápida.
